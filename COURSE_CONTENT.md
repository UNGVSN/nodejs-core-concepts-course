## Course Overview

Understanding Node.js without any other NPM packages. Learn vital computer science concepts and back-end engineering fundamentals that will help you truly master Node.js.

### What You'll Learn
- Learn vital concepts of back-end engineering
- Truly and deeply understand what exactly Node.js is
- Master Node.js as it is without using any other NPM packages
- Understand important operating systems concepts
- Learn how to directly deal with binary data
- Learn the most important Networking concepts
- Deeply understand what HTTP really is and how it works
- Get a deep understanding of the File System
- Master Streams for highly performant applications
- Learn compression to enhance network application speed
- Understand Unix and open up new possibilities
- Master Multi-Threading in Node.js
- Learn cutting-edge cryptographic tools

### Prerequisites
- At least one year of programming experience
- A good knowledge of JavaScript
- A little bit of experience with Express.js and HTML (helpful but not required)
- CPU with at least 4 cores (recommended for Multi-Threading section)
